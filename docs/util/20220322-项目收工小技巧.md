# 项目收工小技巧

## pip 清华源

```shell
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```



## 代码美化

```shell
pip install black
python -m black fileName
python -m black folderName
```

C 和 Java

```shell
clang-format ./main.cpp
clang-format -i ./main.cpp
```



## 依赖包导出

```shell
# https://pypi.org/project/pipreqs/
pip install pipreqs
pipreqs ./ --force
```
