# 设置GitHub为网页



安装 Python 依赖包

```
mkdocs==1.2.2
mkdocs-material==7.1.11
mkdocs-material-extensions==1.0.1
mike==1.0.1
```



## 新建本地项目 myblog

```shell
mkdocs new myblog
```



## 新建GitHub项目 myblog 并 clone 到本地

移入本地文件，并push到远程仓库

新建gh-pages分支

```
git checkout -b gh-pages
```

编译文件

```shell
mkdocs build
```

删除除site之外的文件并将site文件夹的文件更新到root中。并将项目文件夹更新到该分支。

此时查看项目的 setting Pages 即可以看到对应网页地址。
